# sensors-uno-csharp
`sensors-uno-csharp` is a simple template for accesing sensors in uno platform.

These are included for now.
* `Accelerometer`
* `OrientationSensor`
* `Magnetometer`
* `Compass`
* `OrientationSensor`
* `Geolocator`

I would be appliciated if someone can add more sensors.
